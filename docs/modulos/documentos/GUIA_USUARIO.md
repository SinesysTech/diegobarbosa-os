# Guia do Usuário - Editor de Documentos

Este guia explica como usar o Editor de Documentos do Sinesys para criar, editar e compartilhar documentos.

---

## Acessando o Editor

1. Faça login no Sinesys
2. No menu lateral, clique em **Serviços** > **Editor de Documentos**
3. Você verá a lista de seus documentos

---

## Criando um Documento

### Método 1: Documento em Branco
1. Clique no botão **"Novo Documento"**
2. Digite um título para o documento
3. (Opcional) Escolha uma pasta para salvar
4. Clique em **"Criar"**

### Método 2: A partir de um Template
1. Clique no botão **"Novo Documento"**
2. Clique em **"Usar Template"**
3. Navegue pelos templates disponíveis
4. Clique no template desejado
5. O documento será criado com o conteúdo do template

---

## Editando um Documento

### Barra de Ferramentas

A barra de ferramentas no topo do editor oferece:

| Ícone | Função | Atalho |
|-------|--------|--------|
| **B** | Negrito | Ctrl+B |
| *I* | Itálico | Ctrl+I |
| <u>U</u> | Sublinhado | Ctrl+U |
| ~~S~~ | Riscado | Ctrl+Shift+S |
| H1/H2/H3 | Títulos | Ctrl+1/2/3 |
| Lista | Lista com marcadores | - |
| Lista Numerada | Lista ordenada | - |
| Checklist | Lista de tarefas | - |
| Tabela | Inserir tabela | - |
| Imagem | Inserir imagem | - |
| Link | Inserir link | - |
| Código | Bloco de código | - |

### Formatação de Texto

1. **Selecione o texto** que deseja formatar
2. **Clique no botão** correspondente na barra de ferramentas
3. Ou use o **atalho de teclado**

### Inserindo Imagens

1. Clique no ícone de **Imagem** na barra de ferramentas
2. Escolha uma das opções:
   - **Upload**: Selecione um arquivo do seu computador
   - **URL**: Cole o link de uma imagem online
3. A imagem será inserida no documento

### Inserindo Tabelas

1. Clique no ícone de **Tabela**
2. Escolha o número de linhas e colunas
3. A tabela será inserida
4. Clique em uma célula para editar
5. Use Tab para navegar entre células

### Mencionando Usuários

1. Digite **@** seguido do nome do usuário
2. Uma lista de sugestões aparecerá
3. Selecione o usuário desejado
4. O usuário será notificado

---

## Salvamento Automático

O documento é salvo automaticamente a cada 2 segundos de inatividade.

Observe o indicador no canto superior direito:
- **"Salvando..."** - O documento está sendo salvo
- **"Salvo"** - O documento foi salvo com sucesso
- **"Erro"** - Ocorreu um erro ao salvar (verifique sua conexão)

---

## Organizando com Pastas

### Criando uma Pasta

1. Na lista de documentos, clique em **"Nova Pasta"**
2. Digite o nome da pasta
3. Escolha o tipo:
   - **Comum**: Visível para todos
   - **Privada**: Apenas você pode ver
4. (Opcional) Escolha uma cor e ícone
5. Clique em **"Criar"**

### Movendo Documentos

1. Na lista de documentos, clique nos **três pontos** ao lado do documento
2. Selecione **"Mover para..."**
3. Escolha a pasta de destino
4. Clique em **"Mover"**

### Navegando entre Pastas

- Clique em uma pasta para abri-la
- Use o **caminho de navegação** (breadcrumb) no topo para voltar
- Clique em **"Início"** para voltar à raiz

---

## Compartilhando Documentos

### Compartilhar com um Usuário

1. Abra o documento que deseja compartilhar
2. Clique no ícone de **Compartilhar** (ícone de pessoas)
3. Digite o nome ou email do usuário
4. Escolha a permissão:
   - **Visualizar**: O usuário só pode ler
   - **Editar**: O usuário pode modificar o documento
5. Clique em **"Compartilhar"**

### Gerenciando Compartilhamentos

Na janela de compartilhamento, você pode:
- Ver quem tem acesso ao documento
- Alterar permissões clicando no dropdown
- Remover acesso clicando no **X**

### Quem Pode Compartilhar?

Apenas o **criador** do documento pode:
- Compartilhar com novos usuários
- Alterar permissões
- Remover compartilhamentos

---

## Histórico de Versões

O sistema guarda automaticamente as versões do seu documento.

### Visualizando Versões

1. No editor, clique no menu **"..."** (mais opções)
2. Selecione **"Histórico de Versões"**
3. Veja a lista de versões com data e autor

### Restaurando uma Versão

1. Na lista de versões, encontre a versão desejada
2. Clique em **"Visualizar"** para ver o conteúdo
3. Se for a versão correta, clique em **"Restaurar"**
4. Confirme a restauração
5. O documento atual será substituído pela versão selecionada

**Nota**: A versão atual será salva antes da restauração.

---

## Lixeira

Documentos deletados vão para a lixeira e podem ser recuperados.

### Acessando a Lixeira

1. No menu de documentos, clique em **"Lixeira"**
2. Veja os documentos deletados

### Restaurando um Documento

1. Na lixeira, encontre o documento
2. Clique em **"Restaurar"**
3. O documento voltará para a pasta original

### Deletando Permanentemente

1. Na lixeira, clique em **"Deletar Permanentemente"**
2. Confirme a ação
3. **Atenção**: Esta ação não pode ser desfeita!

**Nota**: Documentos na lixeira são automaticamente deletados após 30 dias.

---

## Exportando Documentos

### Exportar para PDF

1. No editor, clique no menu **"..."**
2. Selecione **"Exportar como PDF"**
3. O arquivo PDF será baixado

### Exportar para Word (DOCX)

1. No editor, clique no menu **"..."**
2. Selecione **"Exportar como Word"**
3. O arquivo DOCX será baixado

---

## Chat do Documento

Cada documento tem um chat integrado para colaboração.

### Abrindo o Chat

1. No editor, clique no ícone de **Chat** (balão de conversa)
2. O painel de chat abrirá à direita

### Enviando Mensagens

1. Digite sua mensagem no campo de texto
2. Pressione **Enter** ou clique em **Enviar**
3. A mensagem aparecerá para todos que têm acesso ao documento

---

## Busca Rápida (Command Menu)

Use o atalho **Ctrl+K** (ou **Cmd+K** no Mac) para abrir a busca rápida.

### Funcionalidades

- **Buscar documentos**: Digite o nome do documento
- **Criar documento**: Selecione "Novo Documento"
- **Abrir template**: Navegue pelos templates
- **Ações rápidas**: Acesse funções do sistema

---

## Templates

### O que são Templates?

Templates são documentos modelo que podem ser reutilizados para criar novos documentos com estrutura predefinida.

### Usando um Template

1. Ao criar um novo documento, clique em **"Usar Template"**
2. Navegue pelas categorias ou busque pelo nome
3. Clique no template desejado
4. Um novo documento será criado com o conteúdo do template

### Criando seu Próprio Template

1. Crie ou abra um documento com a estrutura desejada
2. No menu **"..."**, selecione **"Salvar como Template"**
3. Dê um nome e descrição
4. Escolha a visibilidade:
   - **Público**: Todos podem usar
   - **Privado**: Apenas você pode usar
5. (Opcional) Escolha uma categoria
6. Clique em **"Salvar"**

## Problemas Comuns

### "Não consigo salvar o documento"

- Verifique sua conexão com a internet
- Tente recarregar a página
- Se o problema persistir, copie o conteúdo antes de fechar

### "Não encontro meu documento"

- Use a busca rápida (Ctrl+K)
- Verifique se não está na lixeira
- Verifique os filtros de pasta

### "Não consigo editar um documento compartilhado"

- Verifique sua permissão (deve ser "Editar")
- Peça ao criador para alterar sua permissão

### "A imagem não aparece"

- Verifique se o arquivo não excede 50MB
- Verifique o tipo de arquivo (JPG, PNG, GIF, WebP)
- Tente fazer upload novamente
